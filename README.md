# Personal-Accounting-System
## chapter 1 项目介绍
这是一个个人记账系统，用python和SQL写的。首先，需要新建一个数据库mySQL，我用的是Navicicat Premium 16，试用期16天左右。建完数据库，需要把代码里的用户名和密码改成新建数据库的用户和密码。然后运行代码就可以了。python部分用到了模块导入，OOP，异常处理，条件判断与循环。SQL部分用到了数据库连接，DDL，DML，查询特性（条件查询，聚合函数，分组统计）。

## chapter 2 项目功能
**1.用户管理：**
（1）用户登录：允许输入用户名和密码，系统判断用户存在，并且密码正确，则提示登录成功，并展示系统的功能菜单；登录失败则提示用户输入的用户名或密码不正确
（2）用户注册：显示注册新用户，允许用户输入新的用户名和密码，系统判断输入的新用户不与已有用户名相同，则显示注册成功；如果输入已经存在的用户名，则提示用户已存在；如果密码为空，则不允许注册
**2.数据输入：**
  允许用户输入每一笔收入或支出，分开存储收入记录和支出记录。每条收入记录或支出记录均包括用户名、金额、类型、日期和备注字段。类型包括医疗、饮食、交通、购物、其他。
**3.记账管理：**
  （1）允许用户按日期段和类型（收入或支出）查询。用户可输入起始日期和终止日期以及是收入还是支出，系统查询从起始日期到终止日期时间段内的收入或支出记录。
   （2）允许用户修改和删除指定的收支记录。
**4.数据统计：**
 （1）可以按指定日期段内分别对收入和支出数据的金额汇总统计
（2）可以对每个类型下的收入和支出数据的金额分别汇总统计
**P.S.用户交互采用命令行方式**
